### Commit Message

|   type    |                    Explain                    |
| :-------: | :-------------------------------------------: |
|   feat:   | 어떤 특정 어플리케이션에 더할 새로운 feature  |
|   fix:    |              어떤 오류 해결(fix)              |
|  style:   |     스타일과 연관된 feature나 업데이트들      |

|   docs:   |             문서화에 관한 모든 것             |
|  chore:   |              정규 코드 유지보수               |
