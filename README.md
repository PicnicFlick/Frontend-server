## 🔖 소개

**PicnicFlick**은 '스마트 돗자리 대여 서비스'로 돗자리 부재로 인한 불편함 해결, 무인 기기를 통한 간편한 사용 및 반납 시스템을 제공하는 플랫폼입니다.🌟
<br>

## 📚 기술스택

| 분야           | 사용 기술                       | 비고 |
| -------------- | ------------------------------- | ---- |
| FrontEnd       | React, javascript               |
| BackEnd        | Spring, java                         |
| Database       | MySQL             |
| Cloud Services | AWS EC2, AWS Cloudfront, AWS S3 |
| TOOL           | VSCODE, Notion           |
| DESIGN         | Figma                           |
